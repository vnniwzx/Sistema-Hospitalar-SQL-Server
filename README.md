Sistema Hospitalar SQL Server

Projeto de banco de dados hospitalar desenvolvido utilizando SQL Server para praticar modelagem relacional, criação de tabelas e consultas SQL.

🚀 Funcionalidades
Cadastro de pacientes
Cadastro de médicos
Controle de consultas
Solicitação de exames
Relatórios e consultas SQL
Relacionamento entre tabelas utilizando Foreign Keys

🛠️ Tecnologias utilizadas
SQL
SQL Server
SQL Server Management Studio

📚 Conceitos aplicados
Modelagem relacional
Foreign Keys
Joins
Consultas SQL
Filtros e relatórios
Relacionamento entre tabelas

▶️ Como executar o projeto
1. Instale o SQL Server
Baixe e instale:
SQL Server
SQL Server Management Studio (SSMS)
2. Clone o repositório
git clone https://github.com/vnniwzx/Sistema-Hospitalar-SQL-Server.git
3. Entre na pasta do projeto
cd Sistema-Hospitalar-SQL-Server
4. Abra o SQL Server Management Studio (SSMS)
Abra o SSMS e conecte-se ao servidor local.
5. Abra o arquivo hospital.sql
No SSMS:
Clique em File
Clique em Open
Clique em File...
Selecione o arquivo hospital.sql
6. Execute o script
Clique em Execute ou pressione F5.
